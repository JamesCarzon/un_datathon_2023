# un_datathon_2023

This repo contains the work of the Scotty XD team for the United Nations Datathon 2023.

# Usage

The work in this repo depends on the following requirements:
* R libraries
    + reticulate
    + tidyverse
    + ncdf4
    + dplyr
    + melt
    + ggplot2
    + RColorBrewer (optional)
* Python packages
    + numpy
    + matplotlib
    + pandas
    + scipy
    + scikit
